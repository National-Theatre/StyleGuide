Testing display of HTML elements - H1 to start

This set of elements tested is not exhaustive in any sense. Selected elements are those which will usually appear in a standard web document. The appearance in this document should lead the display in default desktop display.

Design will affect the display, size and visuals of elements in a final web document, but this should serve as a starting point for both build and design, looking to base any changes on this stylesheet, to maintain consistency where possible.

The guidelines also apply to the structure of a HTML document. Most opinion on structure is derived from the HTML5 spec - which is subject to change so check at http://html5doctor.com/element-index/ or http://dev.w3.org/html5/spec/single-page.html
